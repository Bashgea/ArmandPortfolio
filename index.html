About Me

I am a Computer Science student who likes turning ideas into real projects. On the web side, I use Java with Spring Boot and Python with Flask for back end APIs, React for the front end, and MySQL for data. I have built a campus navigator, a shopping site with machine learning features, and a blockchain raffle project where I worked on raffle creation, random winner selection, and testing.

I like practical machine learning with scikit learn and XGBoost. I focus on clear metrics and faster results, and I care about performance and accessibility so apps are easy to use.

Outside class, I help students who are new to programming learn to code, contribute to open source projects like pandas_ta, and stay active with ColorStack and NSBE. When I am off the computer, I play soccer with friends, go to the gym, and I have recently started taking pictures of nature.

Augsburg University Navigator

The School Navigator helps Augsburg students find walking routes between campus buildings by combining OpenRouteService and OpenStreetMap data. A Flask backend relays user start points and destinations to the ORS API and returns accurate directions.

My role: Designed the backend and database layer, created the SQLite schema for building coordinates, and built the /navigate endpoint that connects the Flask app with OpenRouteService.

Built with Ayden Lee.
